# What is Docker?

Docker is an open-source platform that automates the deployment, scaling, and management of applications by using containers.

## Core Concepts

- **Containers:**

Lightweight, standalone packages containing everything needed to run an application.

- **Images:**

Read-only blueprints used to create and launch containers.


- **Dockerfile:**

A text script containing instructions to build a Docker image.


- **Docker Hub:** A cloud-based registry for sharing and downloading container images.
